## Описание программы
*"Movie Library — это GUI‑приложение для хранения информации о фильмах. Позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга, отображать их в таблице, фильтровать по жанру и году, а также сохранять и загружать данные в формате JSON.

## Требования для запуска
Что нужно установить на компьютер:

Python 3.10 или выше

Библиотеки: tkinter (обычно встроен в Python), json (встроен)

## Как запустить
bash
git clone https://github.com/твой-логин/movie-library.git
