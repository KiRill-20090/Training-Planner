## Описание программы

*Training Planner — это GUI‑приложение для планирования и учёта спортивных тренировок. Позволяет добавлять тренировки с указанием даты, типа и длительности, фильтровать записи, а также сохранять и загружать данные в формате JSON.*

---

## Требования для запуска

Что нужно установить на компьютер:

* Python 3.10 или выше
* Библиотеки: `pip install tkinter` (обычно встроен в Python), `json` (встроен)
